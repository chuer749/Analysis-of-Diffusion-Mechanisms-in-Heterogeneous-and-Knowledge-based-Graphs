# Analysis-of-Diffusion-Mechanisms-in-Heterogeneous-and-Knowledge-based-Graphs
基于零售异质图的传播机制与决策分析系统 (v0.1)
本项目是一个针对大规模零售异质图（包含 19 万+ 店铺实体）的交互式可视化决策平台。系统通过分析商品、品类、店铺与环境（天气）之间的传播机制，为零售决策提供数据驱动的可视化支持。

核心功能
大规模子图动态采样：支持从 1800+ 店铺中秒级检索特定 ID 的局部拓扑网络，解决海量数据下的渲染瓶颈。

多维异质知识关联：融合了结构化的销售数据（店铺-商品）、层级化的品类知识（商品-品类）以及非结构化的环境因素（店铺-天气）。

传播机制分析：可视化展示折扣力度、销量波动等信号如何在异质图中跨节点类型传播。

决策仿真模拟：通过交互式滑块实时调整折扣敏感度等权重参数，观察图拓扑结构的动态演变。

技术背景：传播机制分析
在本项目构建的异质图中，信息通过以下三条路径进行传播与耦合：

直接销售路径：店铺 (Store) -> 商品 (Product)，反映单店的经营表现。

知识层级路径：商品 (Product) -> 品类 (Category)，体现跨商品的品类协同效应。

环境影响路径：店铺 (Store) -> 天气 (Weather)，揭示外部宏观环境对局部零售生态的影响。

开发者说明
数据量级：1800+ 个店铺节点，支持精准 ID 搜索。

节点命名规范：商品节点已统一重命名为 商品_xx，增强业务可读性。
